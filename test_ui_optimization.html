🔍 测试步骤

1. 打开游戏页面 (index.html)
- 检查血量显示是否包含"血量:"标签
- 检查右侧信息面板是否显示"剩余怪物"

2. 开始游戏
- 观察血量显示格式是否正确
- 观察剩余怪物数量是否实时更新

3. 游戏过程中
- 血量变化时标签是否保持
- 怪物被击败时数量是否减少
- 新怪物激活时数量是否更新

4. 关卡切换
- 新关卡开始时怪物数量是否重置
- 血量显示是否保持正确格式
