🔧 子弹射程修复测试

📋 修复内容

✅ 修复: 子弹射程越来越短的问题
问题: 子弹打着打着越来越短，然后就消失不见
原因: 边界检查过于严格，子弹射程太短
修复: 大幅增加子弹的射程边界

⚠️ 测试要点
1. 子弹是否能正常飞行足够远的距离
2. 子弹射程是否稳定，不会越来越短
3. 子弹是否在合理距离后消失

🔍 边界检查修复对比

修复前

上边界: y + height < -100
下边界: y > 2660
左边界: x + width < -100
右边界: x > 1540
问题: 边界太严格，子弹射程太短
射程: 约1380像素 (0.575秒)

修复后

上边界: y + height < -2000
下边界: y > 3560
左边界: x + width < -1000
右边界: x > 2440
效果: 边界宽松，子弹射程大幅增加
射程: 约3280像素 (1.37秒)

📊 射程计算

子弹参数:
- 速度: 2400像素/秒 (向上)
- 玩家初始Y: 1280像素
- 子弹高度: 50像素

修复前射程:
- 从1280到-100: 1380像素
- 飞行时间: 1380 ÷ 2400 = 0.575秒
- 问题: 射程太短，感觉子弹越来越少

修复后射程:
- 从1280到-2000: 3280像素
- 飞行时间: 3280 ÷ 2400 = 1.37秒
- 效果: 射程增加2.4倍，子弹能飞行更远

实际游戏效果:
- 子弹能击中更远的敌人
- 子弹射程稳定，不会越来越短
- 射击体验更流畅
